# 🚦 Smart Traffic Monitoring & Alert System

## 📌 Overview
This project simulates a real-time traffic monitoring system that analyzes congestion across multiple zones and provides intelligent alerts along with predictive insights. The system is designed to demonstrate how traffic data can be processed, classified, and visualized in an interactive dashboard.

---

## 🎯 Key Features
- 🔄 Real-time traffic simulation (auto-refresh every 5 seconds)
- 🚦 Congestion classification (High, Moderate, Smooth)
- 🚨 Alert system for critical traffic conditions
- 🔮 Predictive insights based on delay trends
- 🗺 Interactive map visualization with live markers
- 📊 Clean dashboard UI with structured layout

---

## 🧠 System Logic
The system follows a rule-based approach:

1. Traffic delay is randomly generated for each zone
2. Delay values are analyzed and categorized:
   - High Traffic → Severe congestion
   - Moderate Traffic → Increasing load
   - Smooth → Normal flow
3. Alerts are triggered for non-smooth conditions
4. A simple forecasting model predicts upcoming traffic conditions
5. Data is refreshed periodically to simulate real-time monitoring
